・ターゲット

特別な日に大切な人とのいい思い出を作りたい人。
サプライズが好きな人。
デートで失敗するのが怖い人。　　　　　　　
自分では思いつかないような変化球なデートをしたい人。

・解決する悩み

店選び、服の選定、プレゼントの選定、女心への寄り添い、デートの失敗率の減少

・主要機能

入力項目（場所、時間、予算、性別、年齢、身長、食の好み、趣味、関係性、共通点、雰囲気、大切にしているものや価値観）

出力項目（場所の提示、服装の提案、髪型の提案、プレゼントの提案）

・こだわりポイント

3つの大まかなデートプランを提案し、他のお客様の成功確率や似ている性格の人のデートプランを閲覧や比較ができる。成功したか失敗したかなどを報告すれば成功確率やその人にあった修繕ポイントを分析提示をする。心理学を用いて解説や失敗しないポイントを随所に提示する。
