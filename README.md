# SmartDate Planner (仮)

AIがあなたのデートをパーソナライズしてプロデュースするWebアプリです。

## 🎯 ターゲット
* 大切な日に最高の思い出を作りたい人
* サプライズの内容や服装選びで失敗したくない人
* 自分の思いつかないような変化球なデートをしたい人

## ✨ 主な機能
* **AIプランニング:** 予算、食の好み、関係性など12項目から最適なプランを提案。
* **トータルコーデ:** 当日の服装や髪型、プレゼントまでAIがアドバイス。
* **成功確率分析:** デート後の報告機能と、成功確率の分析・提示。

## 🛠 使用技術
* HTML / CSS / JavaScript
* Gemini API (AI)
* GitHub / Vercel
